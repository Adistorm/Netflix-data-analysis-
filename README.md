# Netflix Data Analysis 📊

## Project Overview

This project analyzes the Netflix dataset to explore trends in movies and TV shows available on the platform. The goal is to understand content distribution, popular genres, release trends, and other insights using data analysis and visualization techniques.

## Objectives

* Analyze the distribution of movies and TV shows on Netflix.
* Identify the most frequent genres in the dataset.
* Find the most popular movies based on popularity scores.
* Explore the year with the highest number of releases.
* Visualize patterns using data visualization tools.

## Dataset

The dataset used for this project contains information about Netflix titles including:

* Title
* Genre
* Release Year
* Popularity
* Vote Count
* Type (Movie / TV Show)

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Analysis Steps

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Conclusion and Insights

## Key Insights

* **Drama** is the most frequent genre in the dataset.
* Around **25.5% of titles have high popularity votes**.
* **Spider-Man: No Way Home** has the highest popularity among movies.
* The year **2020 recorded the highest number of movie releases**.
